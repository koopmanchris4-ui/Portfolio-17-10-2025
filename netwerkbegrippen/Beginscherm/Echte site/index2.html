Firewall

Een Firewall bepaalt of verkeer van buiten het netwerk veilig genoeg is om binnen te komen en dient dus als een soort muur voor onveilig verkeer. Dit kan zowel een apparaat als een programma zijn. Zo'n Firewall wordt vaak gebruikt bij een bedrijfsnetwerk.

Bron: Digitaltrustcenter.nl


DHCP

DHCP staat voor Dynamic Host Configuration Protocol. Het is een netwerkbeheerprotocol dat automatisch IP-adressen aan apparaten geeft die op een netwerk aangesloten zijn. Doordat dit automatisch gaat, is het toewijzen van IP adressen niet foutgevoelig en gaat het bliksemsnel.

Bron: Wikipedia.nl, Nordvpn.com


DNS

DNS staat voor domain name system. Het is een systeem dat domeinnamen (zoals Youtube.com of Google.com) omzet in IP adressen die computers begrijpen. Dit is belangrijk, omdat we hierdoor geen lange IP-adressen moeten onthouden. Inplaats daarvan zijn het simpele website domeinen.

Bron: domeinnaam.com, computerlesvoorbeginners.nl


DDNS

DDNS staat voor dynamic domain name system. Dit is eigenlijk een soort aanvullen op DNS wat ervoor zorgt dat de IP-adressen die aan domeinnamen zijn gekoppeld automatisch worden bijgewerkt. Hierdoor hoef je je geen zorgen te maken over de wijzigingen van het IP-adres.

Bron: aws.com, ovh.com


Poort

Een poort is een virtueel toegangspunt op een apparaat dat gebruikt wordt voor communicatie tussen apparaten en computerprogramma's. Voorbeelden van verschillende poorten zijn: Webpservers, e-mailservers of andere diensten.

Bron: computerhulpentips.nl
